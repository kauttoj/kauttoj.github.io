# Images Directory

## Required Images

Please add the following images to this directory:

1. **my-avatar.png** - Your professional photo (recommended: 150x150px or larger, square format)
   
2. **project-placeholder.jpg** - Default image for projects without specific images (400x300px recommended)

3. **logo.ico** - Favicon for the browser tab

## Placeholder Images

Temporary placeholder images have been created. Please replace them with your actual images for the best results.

## Image Recommendations

- **Avatar**: Use a professional headshot with good lighting
- **Project images**: Use screenshots or representative images of your work
- **Format**: PNG for avatar (supports transparency), JPG for projects
- **Size**: Keep images optimized for web (under 500KB each)
